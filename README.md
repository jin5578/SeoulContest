
# FOODSTAMP

> '2018년 서울시 앱 공모전 출품작’ 

음식의 사진을 찍고 가게, 메뉴 정보를 사진위에 입력할 수 있는 포토샵 기능과 다른 사용자들과 공유할 수 있는 어플리케이션입니다.


## 개발환경

* 개발툴
    
    * Android Studio
    
* 언어
    
    * Kotlin

* Code Architecture

    * MVP Pattern
    
* 라이브러리

    * Glide
    * RecyclerView
    * Retrofit2

* 서버

    * MySQL
    * SQLite
    * PHP
